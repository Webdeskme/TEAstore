What is TEAhouse

TEAhouse is a new linux distribution made from a hibrid of Debian and Nodejs that should run well on most pcs. It was designed to allow users to easily take control over there apps and operating system.

CUSTOMIZE AND CHANGE TEA APPS

You can build your own apps or custimize others at anytime because the code is always open source and accessible.

Light & Fresh

TEAroom apps share one dependency library that is always kept up to date. This means app developers can consitrate on just maintaining a few dependencies. This keeps each app slim and up todate. Apps can still include there own dependincies if they wish but it is encouraged to use the dependencies provided. Either way the code is always open for you to tinker with.

TEA apps HTML and Javascript

Our OS and apps are javascript so you can easily make app or tweek any TEA app on your system. That being said you can still download the apps you know and love wether they are Debian, Flatpak, Snap, or AppImage.

So many reasones to try today!

  • Light as a feather

    Out of the box this system is light. TEAhouse is built off of a minimal install of Debian. the TEAroom apps all share dependencies wich help reduse the size of the system. As this distro matures it is our goal to keep optimizing TEAhouse to be light while providing a full experiance.

  • Cool Desktop

    The desktop screen is html with javascript so you can get many cool custimizable background screens.

  • Easy Coding

    HTML, Javascript, CSS, and Nodejs. Thats all you need to know!

  • Music to my ears

    Quickly be able to navigate your apps.

  • Always Be Backed Up

    TEAhouse will allow you to back up your system fully to a flashdrive so that you can rebuild your system if need be. TEAhouse also provides more troditional methods of backup as well.

  • Sharing is Caring

    Share anything of local wifi by just adding it to the share directory in your home folder.
